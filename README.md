# Plexi Materials

Welcome to the **Plexi Materials** repository! This repository acts as the central storage and index for all the study materials, notes, PDFs, PowerPoint presentations, question banks, and syllabuses used in the Plexi platform.

## 📂 Key Files

* `manifest.json`: The core index that categorizes all available study materials by semester, subject, and format. It serves as the single source of truth for the Plexi Hub frontend.
* `indexed_files.json`: A tracked list of all files that have been successfully parsed and processed into the vector database (LlamaIndex) for the AI assistant's Retrieval-Augmented Generation (RAG) capabilities.

## 🛠 Adding New Materials

When you want to add new materials to the Plexi platform:
1. **Create Issue:** Create a Issue with [Upload Study Material](https://github.com/KunalGupta25/plexi-materials/issues/new?template=upload-material.yml) Template and fill the details.
2. **Upload the file:** Attach the file in the issue and Submit it.

After submission, the file will be automatically processed and added to the Plexi platform.
